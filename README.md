# Thalassemia Chatbot
![Python3](https://img.shields.io/badge/Python3-Language-blue?logo=python)
## BIO 101 - Cell Biology and Public Health - Final Project
## Group Members: Anand Kumar, Nimrah Javed, Umme Salma, Shoaib Khursheed
A basic chatbot that gives information about the thalassemia disease and answers basic questions related to the disease. The chatbot was developed using the RASA framework which is a Python3 chatbot management framework.
